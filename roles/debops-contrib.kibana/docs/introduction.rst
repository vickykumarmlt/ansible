Introduction
============

.. include:: includes/all.rst

The ``debops-contrib.kibana`` role allows you to manage and configure
`Kibana <https://www.elastic.co/products/kibana/>`, a data visualization and
analytics dashboard for `Elasticsearch <https://www.elastic.co/products/elasticsearch>`
It provides a Web interface on top of the content indexed on a Elasticsearch
cluster.


Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.0.0``. To install it, run:

.. code-block:: console

   ansible-galaxy install debops-contrib.kibana
